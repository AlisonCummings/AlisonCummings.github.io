Set up a webpage in GitHub

Create a Repository

Register for Github at GitHub.com and Log in

Create a Repository at username.github.io by locating the New button, or by navigating to github.com/new while logged in

Use your 'username.github.io' in the "Repository Name *" field

Scroll down and click "Create Repository"

Upload Files

Drag and Drop or browse your computer for HTML, CSS and all associated files for the page

Commit Your Changes

Page will then be viewable at your username.github.io
